# Inter Font Files

This directory should contain Inter font files for English (LTR) display.

## Required Files:

Please download the following font files from https://github.com/rsms/inter/releases or https://rsms.me/inter/ and place them in this directory:

- Inter-Thin.ttf (font-weight: 100)
- Inter-ExtraLight.ttf (font-weight: 200)
- Inter-Light.ttf (font-weight: 300)
- Inter-Regular.ttf (font-weight: 400)
- Inter-Medium.ttf (font-weight: 500)
- Inter-SemiBold.ttf (font-weight: 600)
- Inter-Bold.ttf (font-weight: 700)
- Inter-ExtraBold.ttf (font-weight: 800)
- Inter-Black.ttf (font-weight: 900)

## Download Links:

1. Visit: https://github.com/rsms/inter/releases
2. Download the latest release (usually a ZIP file)
3. Extract the files from the `docs/font-files/` directory
4. Copy the TTF files listed above to this directory

Alternatively:
- Visit: https://rsms.me/inter/
- Click "Download" to get the complete font family
- Extract and copy the TTF files to this directory
